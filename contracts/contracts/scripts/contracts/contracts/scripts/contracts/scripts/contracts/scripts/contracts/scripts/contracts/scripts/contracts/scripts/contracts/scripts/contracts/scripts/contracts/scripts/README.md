## Deploy in 2 min (mobile)
1. Open https://remix.ethereum.org  
2. Plugin manager → enable "Deploy & Run" → Environment → "Injected Provider" → connect MetaMask (Base Sepolia test-net)  
3. File explorer → create `deploy-and-call.js` → paste contents  
4. Edit line 4: insert your test-net private key (0x...)  
5. Run → wait ~30 s  
6. Copy the two contract addresses printed and the tx hash → screenshot  
7. Paste screenshot into root README → push
